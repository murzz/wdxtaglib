WDXTagLib is a content plugin for Total Commander which uses TagLib library
for reading and editing the meta-data of several popular audio formats.

Currently it supports both ID3v1 and ID3v2 for MP3 files, Ogg Vorbis comments
and ID3 tags and Vorbis comments in FLAC, MPC, Speex, WavPack TrueAudio,
WAV, AIFF, MP4 and ASF files. 

For more information visit http://code.google.com/p/wdxtaglib/wiki/Introduction
